# ecommerce-rules-query

电商平台官方规则查询助手。  
用于查询抖音、淘宝、天猫、拼多多、闲鱼、京东、小红书、快手等平台的官方规则、入驻门槛、资质要求、保证金标准、粉丝门槛和违规处罚。

## 这个 Skill 解决什么问题

很多电商新手在开店、带货、上架商品前，最容易遇到的问题不是“怎么运营”，而是：

- 我现在能不能开店？
- 我需要营业执照吗？
- 我卖这个类目要不要许可证？
- 保证金要交多少？
- 多少粉丝才能直播带货？
- 违规后会怎么处罚？
- 平台规则在哪里看？

这些问题属于平台硬性规则，不能只靠经验或模型记忆回答。因为不同平台、不同品类、不同身份的规则差异很大，而且规则会频繁更新。

`ecommerce-rules-query` 的目标是：  
**帮助用户先查清官方规则，再决定下一步怎么操作。**

## 适用场景

适合在以下问题中使用：

- 查询平台开店条件
- 查询直播带货、短视频带货、橱窗带货门槛
- 查询保证金标准
- 查询商品类目资质要求
- 查询营业执照、食品经营许可证等准入要求
- 查询违规处罚、申诉规则
- 对比多个平台的入驻规则
- 帮新手理解电商平台里的常见术语

示例问题：

```text
我想在抖音卖食品，需要什么资质？
拼多多个人开店要多少保证金？
小红书开店需要营业执照吗？
抖音多少粉丝可以直播带货？
淘宝卖美妆有什么资质要求？
快手小店违规会怎么处罚？
