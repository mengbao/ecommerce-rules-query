# 电商术语速查表

输出中涉及这些术语时，按用户等级决定是否解释：
- **L0**：首次出现必须用通俗语言解释
- **L1**：首次出现时简要括号注释
- **L2/L3**：不解释

## 抖音电商

| 术语 | 通俗解释 |
|------|---------|
| 有效粉丝 | ≠ 粉丝总数。抖音过滤僵尸粉和不活跃粉丝，只统计近期有互动的真实活跃粉丝。有效粉丝数通常明显少于粉丝总数。 |
| 橱窗带货（商品橱窗） | 在个人主页展示商品链接，粉丝可点击购买。也叫"小黄车"。 |
| 直播带货 | 通过直播间实时展示和销售商品，观众可直接下单。需满足粉丝门槛（通常200有效粉丝）。 |
| 视频带货 | 在短视频中挂商品链接。需满足粉丝门槛（通常500有效粉丝）。 |
| 图文带货 | 在图文内容中挂商品链接。需满足粉丝门槛（通常1000有效粉丝）。 |
| 小黄车 | 抖音直播间和视频中的商品购物车图标，是橱窗/直播带货的入口。 |
| 精选联盟 | 抖音达人分销平台，商家设佣金后达人可选品推广，成交后获佣金。适合无货源达人。 |
| 抖店 | 抖音电商商家管理后台，开店、上架、发货、售后等都在此操作。 |
| 个人店 | 以个人身份证注册，无需营业执照，可经营类目有限。 |
| 个体店 | 以个体工商户营业执照注册，可经营类目比个人店多。 |
| 企业店 | 以企业营业执照注册，可经营类目最全。 |
| 作者保证金 | 开通橱窗带货需缴纳的保证金（通常500元），退出可退。 |
| 店铺保证金 | 开设抖店需缴纳的保证金，按类目不同（通常500-50000元不等）。 |

## 淘宝/天猫

| 术语 | 通俗解释 |
|------|---------|
| 千牛 | 淘宝/天猫商家后台管理工具（APP+PC端），开店、上架、发货、客服等都在此操作。 |
| DSR评分 | 店铺动态评分，由买家评价得出，含描述相符、服务态度、物流速度三项。影响流量和活动报名。 |
| A类/B类违规 | A类为一般违规（如滥发信息），B类为严重违规（如售假、虚假交易）。B类处罚更重。 |
| 天猫年费 | 天猫店铺每年需缴的技术服务年费（通常3万或6万），达一定销售额可返还。 |

## 拼多多

| 术语 | 通俗解释 |
|------|---------|
| 仅退款 | 买家申请退款不需退货，平台直接退款。拼多多对买家保护力度大。 |
| DSR | 拼多多店铺评分，影响流量分配和活动报名资格。 |
| 百亿补贴 | 拼多多官方补贴活动，商品价格低于市场价，报名需满足严格条件。 |
| 多多进宝 | 拼多多达人分销平台，类似抖音精选联盟。 |

## 小红书

| 术语 | 通俗解释 |
|------|---------|
| 店铺分 | 小红书店铺综合评分（满分5分），直接影响流量倾斜、活动报名和结算账期。低于4.5分会被限制。 |
| 专业号 | 小红书商家/创作者账号类型，开通后才能开店和商业推广。 |
| 薯条 | 小红书内容加热工具，付费推广笔记增加曝光量。 |
| 聚光 | 小红书广告投放平台，用于精准投放广告。 |

## 通用术语

| 术语 | 通俗解释 |
|------|---------|
| 一件代发 | 不需自己囤货，客户下单后由供应商直接发货。适合无资金囤货的新手。常见货源平台：1688。 |
| SKU | 商品最小单位规格，如"红色M码"和"黑色L码"是两个SKU。 |
| SPU | 商品总称，如"某品牌T恤"是一个SPU，下有多个不同颜色尺码的SKU。 |
| 转化率 | 访客中实际下单购买的比例。如100人看商品，5人下单，转化率5%。 |
| 客单价 | 平均每个订单金额。如一天10单共500元，客单价50元。 |
| GMV | 商品交易总额。注意≠实际收入，包含退款和未结算订单。 |
| 退换货率 | 已售商品中申请退货或换货的比例。 |
| 类目准入 | 某些品类需额外资质才能销售。如食品需食品经营许可证。 |
| 技术服务费 | 平台从每笔交易中收取的费用，按成交金额比例收取（1%-5%不等）。 |

## 易混淆术语对照

| 术语A | 术语B | 区别 |
|-------|-------|------|
| 有效粉丝 | 粉丝总数 | 有效粉丝 ≤ 粉丝总数，过滤了僵尸粉 |
| 橱窗带货 | 直播带货 | 橱窗是主页挂链接，直播是实时展示销售 |
| 个人店 | 个体店 | 个人店用身份证，个体店用个体工商户执照 |
| 保证金 | 技术服务费 | 保证金可退，技术服务费是平台抽成不可退 |
| 仅退款 | 退货退款 | 仅退款不退货，退货退款需寄回商品 |
